Question:

Which equation demonstrates the Distributive Property

3 • (8 + 2) = 3 • (2 + 8)
2 • (5 • 17) + 12 = (2 • 5) • 17 + 12

2 • (8 + 9) = 2 • 8 + 2 • 9

4 • (7 + 0) = 4 • 7

Answer #1:

First option isn't answer because it used commutativity inside brackets.

Second option isn't answer because there is * between 5 and 17 and not + than 2 should multiply 5 than add that to 2*17

Third option is indeed answer. It represents distributive property on operations multiplying and summing.

Last option isn't answer because there is missing + 4*0
